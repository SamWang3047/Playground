# Weekly Report (Next.js Day 1)

离线创建的 Next.js 项目骨架，目标是用于 7 天全栈冲刺。

## 1. 安装依赖

```bash
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

打开 <http://localhost:3000>

## Day 1 完成项

- Next.js App Router 基础结构
- Tailwind CSS 配置
- React 组件拆分（Header / ReportCard / ReportForm）
- `useState` + 表单提交（当前为前端模拟）

## 下一步（Day 2）
